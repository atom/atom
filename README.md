## Installing

It will automatically update when a new release is available.

## Building

### Requirements

  * Mountain Lion
  * Boxen (Obviously Atom won't release with this requirement)

### Installation

  1. `gh-setup atom`

  2. `cd ~/github/atom`

  3. `script/build`
